# Veriora

**The Light of Truth**

Veriora is a trust-verification platform designed to help users identify scams, verify digital identities, and make safer decisions online.

## Mission

Help people **verify before they trust**.

## Core Features

* Phone Number Verification
* Website Verification
* UPI Verification
* Community Scam Reports
* Trust Score System
* User Reporting System
* Real-Time Notifications
* Firebase Authentication
* Secure Cloud Database

## Tech Stack

* Kotlin
* Jetpack Compose
* Firebase Authentication
* Cloud Firestore
* Firebase Storage
* Material 3
* MVVM Architecture
* Dependency Injection
